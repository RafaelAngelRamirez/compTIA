#### Operating Systems
---

* What is it? 
	+ User <--> Computer
	+ Software <--> Hardware
	+ Common environment for software to execute in
* Types
	+ Workstation
	+ Server
	+ Mobile
	+ Embedded (Firmware)
	+ Hypervisor
* How does it work? 
	+ HAL
	+ Drivers
* Additional Functionality
	+ Logging/Auditing
	+ Permissions/Access Control
	+ Multitasking
	+ Background Services
	+ Crash Recovery
	+ Application Management
	+ Software Updates
	+ Network Services
